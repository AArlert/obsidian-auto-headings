# 场景 06：前缀 + 后缀（「第1章」式编号）

每级格式的字段顺序为 **前缀 + 序号 + 后缀**（再接序号间隔符 / 标题间隔符）。用「前缀」与
「后缀」配合即可拼出「第1章」「第1节」这类带文字的编号。

**需要的模板配置**（设置面板 → 展开要改的模板 → 网格里逐级填）：

| 级别 | 前缀 | 序号 | 后缀 | 标题间隔符 |
|------|------|------|------|-----------|
| H2 | `第` | 阿拉伯 | `章` | 空格 |
| H3 | `第` | 阿拉伯 | `节` | 空格 |
| H4 起 | 空 | 阿拉伯 | 空 | 空格 |

> 后缀作用于**完整序号**：H3 继承父级时是「第1.1节」，而不是把后缀塞进每一段。

**上述配置下的预期结果**：

- `## 引言` → `第1章 引言`
- `## 正文` → `第2章 正文`
- `### 背景` → `第2.1节 背景`
- `### 现状` → `第2.2节 现状`
- `#### 细分` → `2.2.1 细分`（H4 未配前后缀）

**可复制内容**（复制后，请按上表先把模板配置好，再粘贴观察编号）：

```md
# 文档标题

## 引言

## 正文

### 背景

### 现状

#### 细分
```
