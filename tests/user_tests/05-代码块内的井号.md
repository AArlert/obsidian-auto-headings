# 场景 05：代码块内的 `#` 不被当作标题

围栏代码块（```` ``` ```` 包裹）内部以 `#` 开头的行是代码 / 注释，**不应**被解析为标题，
也不应被编号。用于确认解析器正确识别代码块边界。

**默认模板下的预期结果**：

- `## 真实标题` → `1`
- 代码块里的 `# 这是注释` / `## 也不是` → **原样保留，不编号**
- `## 另一个真实标题` → `2`（代码块内的行不占槽位）

**可复制内容**（注意外层用了 4 个反引号，以便把内层 3 反引号的代码块一并复制）：

````md
# 文档标题

## 真实标题

```python
# 这是注释，不是标题
## 也不是标题
def foo():
    return 1
```

## 另一个真实标题
````
