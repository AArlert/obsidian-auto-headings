- `## 引言` → `1`

- `### 数据来源` → `2.1`
- `### 实验设置` → `2.2`
- `#### 硬件` → `2.2.1`
- `#### 软件` → `2.2.2`
- `## 结论` → `3`（回到 H2，更深级别归零）
