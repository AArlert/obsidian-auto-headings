# Auto Headings（自动标题编号）

[English](README.md) | **简体中文**

为 [Obsidian](https://obsidian.md) 自动编号 Markdown 标题，并且**不断链**——改标题（哪怕连文字一起改），所有指向它的 `[[笔记#标题]]` 引用都会跟着自动更新。

装上即得 `1` / `1.1` / `1.1.1` 多级编号，零配置；再按需定制一切：序号样式、按文件夹选模板、白名单、编号层级区间……界面中英双语，自动跟随 Obsidian 语言。

## 随便改标题，链接自动跟

大多数编号插件只更新数字，标题文字变了引用却不跟。本插件两者都同步。

```
<!-- note.md，改动前 -->
## 1 Foo

<!-- other.md -->
详见 [[note#Foo]]。
```

你改了标题——不只是数字，连文字一起改：

```
<!-- note.md，改动后 -->
## 2 Foobar
```

引用在同一次编辑里自动跟着变：

```
<!-- other.md -->
详见 [[note#Foobar]]。
```

不用手动全库查找替换，锚点不断。（少数边界情况——重复标题、块引用、多级锚点——出于安全考虑不处理，见[说明](#说明)。）

## 亮点

-   **开箱即用**——内置默认模板，编辑笔记即得 H2–H6 的 `1 / 1.1 / 1.1.1` 编号。停止输入后经防抖延迟自动重算写回（延迟可调）。
-   **默认链接安全**——Backlink 同步（默认开启）：只要编号改写了标题文本，库中所有指向它的 `[[笔记#标题]]` 链接都会自动更新，不止编号变化，两次编号之间的纯改名也覆盖。
-   **完全可定制的模板**——每级（H1–H6）可设：前缀、序号样式、序号间隔符、后缀、标题间隔符、继承前级。七种序号样式：阿拉伯、中文（一二三）、带圈（①②③）、大小写字母、大小写罗马。编辑时实时预览。模板级设置：编号层级区间（如只编 H2–H4）、起始编号数字（设 0 得 `0.1.1`）、跳级占位策略、祖先序号渲染（`1.a.①` 提纲式或 `一` + `1.1` 中文书式）。
-   **按文件夹选模板**——路径规则表把文件夹 / 文件映射到具名模板，最具体优先。`/papers` 用学术编号、`/book` 用章节样式，互不干扰。
-   **白名单**——目录、附录、参考文献等结构性标题不编号、不占号。三种匹配方式（全部 / 部分 / 子树）；子树块视为独立结构，其后编号重新开始（如附录之后的新章节）。
-   **清除命令家族**——清除当前文件编号、清理外来 / 手写编号（接管导入文档）、清除全库编号（二次确认）。
-   **尊重你的结构**——插件只增删编号前缀，**绝不**改写标题的 `#` 层级；只解析正在编辑的文件，从不全库扫描。

## 工作原理——以及一件你应该知道的事

编号**写入 Markdown 源文本**（而非渲染层覆盖），因此导出、同步、配合其他工具都不丢失。

为了把自己写的编号与你的正文区分开，插件在每个编号前缀末尾写入一个不可见的 **Word Joiner** 字符（U+2060）：`## 1 ⁠我的标题`。这正是安全性的来源——只是*看起来*像编号的正文（`2024 年度总结`、`API 设计`）不会被误当旧编号吃掉。

对你的影响：

-   该字符不可见，不影响排版、导出与阅读。
-   复制 / 导出的文本会带着它；若用 `grep` 精确检索标题文本，注意它位于序号与标题之间。
-   若手动删编号残留了零散字符，「清除当前文件编号」/「清理非本插件编号」命令可一键清理。

## 快速上手

1. 安装并启用插件（见下）。
2. 打开任意笔记开始编辑——H2 及更深的标题自动获得编号。
3. 打开 **设置 → Auto Headings** 按需定制：
    - **全局设置**：语言、全局自动编号开关、Backlink 同步、防抖延迟。
    - **路径与模板**：路径规则表 + 模板编辑器（实时预览、白名单）。
    - **敏感操作**：三个清除入口。
4. 按文件覆盖：frontmatter 写 `obsidian-auto-headings: true/false` 可强制开 / 关单个文件；「立即重新编号」命令绕过一切开关。

## 安装

**社区插件市场**：设置 → 社区插件 → 浏览 → 搜索“Auto Headings”。

**手动安装**：从[最新 Release](../../releases/latest) 下载 `main.js`、`manifest.json`、`styles.css` 放入 `<库>/.obsidian/plugins/obsidian-auto-headings/`，重载 Obsidian 后启用。

## 命令

| 命令                   | 作用                                           |
| ---------------------- | ---------------------------------------------- |
| 立即重新编号           | 立刻重编当前文件，绕过一切开关                 |
| 清除当前文件编号       | 剥离全部编号前缀（含手写样式）                 |
| 清理非本插件的标题编号 | 只剥**非**本插件写入的编号——用于接管导入的文档 |

## 说明

-   **语言**：插件界面自动跟随 Obsidian 语言（中 / 英），也可在设置中锁定。本 README 有[英文版](README.md)。
-   **Backlink 同步的边界**：为避免误改，以下情况保守跳过——重复标题（同名标题多处）、块引用（`^id`）、多级锚点（`#A#B`）。开启同步不会回溯修复开启之前就已断的链接。也可在**设置 → 全局设置**中关闭。
-   **撤销**：单文件改写是一次编辑器事务，一次 `Ctrl/Cmd+Z` 即可撤销；对*其他*文件的 Backlink 改动不在该事务内。「清除全库」**不在**撤销历史内，请先备份。
-   **移动端**：支持（`isDesktopOnly: false`）。
